# Ambiente de Desenvolvimento

## Objetivo
Padronizar o ambiente de desenvolvimento utilizado pela equipe do projeto ISTOKE.

## Ferramentas utilizadas

- Flutter SDK
- Dart
- Android Studio
- Visual Studio Code
- Git
- GitHub
- Firebase

## Sistema operacional utilizado

- Windows 11

## Configuração inicial

1. Instalar Git
2. Instalar Flutter SDK
3. Instalar Android Studio
4. Instalar Visual Studio Code
5. Clonar repositório GitHub
6. Executar comando:

flutter pub get

7. Executar aplicação:

flutter run

## Organização do desenvolvimento

- Utilização de branches para novas funcionalidades
- Utilização de Pull Requests
- Controle de versionamento com GitHub
- Documentação centralizada na pasta docs

## Ambiente de produção

O ambiente de produção será hospedado em nuvem utilizando Firebase, permitindo integração entre aplicativo móvel e banco de dados em tempo real.

## Ambiente de desenvolvimento

O ambiente de desenvolvimento será utilizado localmente pelos integrantes da equipe para testes e implementação de funcionalidades antes da publicação em produção.
