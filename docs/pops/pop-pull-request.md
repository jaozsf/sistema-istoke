# POP — Abertura de Pull Request

## Objetivo
Padronizar o processo de abertura de Pull Requests no projeto ISTOKE.

## Responsável
Desenvolvedor

## Quando usar
Após finalizar uma funcionalidade.

## Pré-requisitos
- Branch criada
- Commits realizados

## Passo a passo

1. Acessar GitHub
2. Selecionar branch
3. Clicar em "Compare & Pull Request"
4. Adicionar descrição
5. Solicitar revisão
6. Realizar merge após aprovação

## Evidências esperadas
- Pull Request criado
- Revisão registrada

## Critérios de sucesso
- Código integrado corretamente
