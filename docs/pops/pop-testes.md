# POP — Execução de Testes

## Objetivo
Garantir que as funcionalidades do sistema ISTOKE estejam funcionando corretamente antes da publicação.

## Responsável
QA/Testador

## Quando usar
Antes do merge ou publicação de novas versões.

## Pré-requisitos
- Projeto funcionando
- Ambiente configurado

## Passo a passo

1. Executar aplicação
2. Testar cadastro de produtos
3. Testar leitura de QR Code
4. Verificar erros
5. Registrar falhas encontradas

## Evidências esperadas
- Prints dos testes
- Registro de bugs

## Critérios de sucesso
- Funcionalidades funcionando corretamente
