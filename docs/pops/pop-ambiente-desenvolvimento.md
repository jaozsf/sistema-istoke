# POP — Configuração do Ambiente de Desenvolvimento

## Objetivo
Padronizar o ambiente de desenvolvimento utilizado pela equipe ISTOKE.

## Responsável
Equipe de Desenvolvimento

## Quando usar
Ao configurar um novo computador para desenvolvimento.

## Pré-requisitos
- Computador com Windows 11
- Internet

## Passo a passo

1. Instalar Git
2. Instalar Flutter SDK
3. Instalar Android Studio
4. Instalar VS Code
5. Clonar repositório
6. Executar:

flutter pub get

7. Executar aplicação:

flutter run

## Evidências esperadas
- Projeto executando localmente

## Critérios de sucesso
- Ambiente funcionando corretamente
