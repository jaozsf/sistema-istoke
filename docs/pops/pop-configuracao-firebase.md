# POP — Configuração do Firebase

## Objetivo
Padronizar a configuração do Firebase no projeto ISTOKE.

## Responsável
Desenvolvedor Backend

## Quando usar
Durante integração com banco de dados.

## Pré-requisitos
- Conta Google
- Firebase instalado
- Flutter configurado

## Passo a passo

1. Acessar Firebase Console
2. Criar projeto
3. Registrar aplicativo Android
4. Baixar arquivo google-services.json
5. Inserir arquivo no projeto
6. Configurar dependências
7. Executar aplicação

## Evidências esperadas
- Firebase conectado ao aplicativo

## Critérios de sucesso
- Comunicação funcionando corretamente
