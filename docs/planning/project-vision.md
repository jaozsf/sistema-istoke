Visão do Projeto – ISTOKE

📌 Descrição Geral
O ISTOKE é um sistema de controle de estoque baseado na utilização de QR Code, com o objetivo de automatizar o registro de entrada e saída de produtos.
A proposta do sistema é permitir que cada item possua um QR Code único, que será lido por meio de um aplicativo mobile. A partir dessa leitura, as informações serão enviadas para um sistema central, responsável por armazenar e atualizar os dados do estoque.

---

🎯 Objetivo do Projeto
Desenvolver uma solução prática e eficiente para o controle de estoque, reduzindo erros manuais e facilitando o gerenciamento de produtos em tempo real.

---

👥 Público-Alvo
O sistema é voltado principalmente para:
- Pequenos comércios
- Estoques de empresas de pequeno porte
- Usuários que necessitam de controle simples e rápido de produtos

---

⚙️ Funcionalidades Principais
O sistema deverá conter as seguintes funcionalidades:

- Cadastro de produtos
- Geração de QR Code para cada item
- Leitura de QR Code por meio de aplicativo mobile
- Registro de entrada de produtos no estoque
- Registro de saída de produtos
- Armazenamento das informações em banco de dados
- Visualização do estoque atualizado

---

🛠️ Tecnologias Previstas
Para o desenvolvimento do sistema, serão utilizadas as seguintes tecnologias:

- Flutter (desenvolvimento mobile)
- Backend (em definição)
- Banco de dados (em definição)
- Git e GitHub para versionamento

---

📊 Estado Atual do Projeto
O projeto encontra-se em fase inicial, com foco no planejamento, organização da equipe e definição da estrutura do sistema.

Atualmente, está sendo iniciado o desenvolvimento do backend, que será responsável pela lógica e processamento das informações.
