1º Bimestre:

- Desenvolvimento do backend
- Criação do banco de dados

3º Bimestre:
- Desenvolvimento do front-end
- Integração entre sistema e banco de dados

4º Bimestre:
- Testes do sistema
- Correção de erros
- Finalização do projeto
