Desenvolvimento vs Produção – ISTOKE

Para garantir a organização do projeto, foi definida uma separação entre ambiente de desenvolvimento e ambiente de produção.

🛠️ Ambiente de Desenvolvimento

- Utilizado para criação e testes do sistema
- Pode conter erros ou funcionalidades incompletas
- Alterações são feitas continuamente pelos integrantes

🚀 Ambiente de Produção

- Representa a versão final e estável do sistema
- Contém apenas funcionalidades concluídas e testadas
- Utilizado como base para entrega do projeto

🔄 Controle de Versão

O controle entre os ambientes é realizado por meio do GitHub, utilizando commits para registrar alterações e manter o histórico do projeto organizado.
