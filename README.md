# 📦 Sistema ISTOKE

Sistema de controle de estoque desenvolvido como projeto acadêmico do curso de Análise e Desenvolvimento de Sistemas.

## 📌 Objetivo

O objetivo do projeto é criar um sistema capaz de gerenciar produtos em estoque, permitindo controle de entrada e saída, além de organização e visualização dos dados, com suporte a interface mobile.

## ⚙️ Funcionalidades

- Cadastro de produtos  
- Controle de entrada e saída  
- Organização do estoque  
- Integração com QR Code (em desenvolvimento)  
- Interface mobile com Flutter  

## 🧱 Estrutura do Projeto

- /backend → responsável pela lógica do sistema  
- /frontend → interface web  
- /mobile → aplicação mobile desenvolvida em Flutter  

## 🚀 Tecnologias Utilizadas

- Python / Node.js (backend)  
- HTML, CSS e JavaScript (frontend)  
- Flutter (mobile)  
- Git e GitHub (versionamento)
